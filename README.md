# E-commerce Sales Analysis
**Tools:** Python (Pandas, Matplotlib, SQL), Power BI  
**Goal:** Analyze online store performance and identify key revenue drivers.

## Workflow
1. Data cleaning & feature creation
2. SQL queries for sales aggregation
3. Power BI dashboard for visualization
4. Business Recommendations/Insights

## Files
- `ecommerce_cleaning.ipynb` — full notebook
- `Ecommerce_Clean.csv` — prepared dataset
- `Ecommerce_Dashboard.pbix` — Power BI dashboard

## Note
The dataset used in this notebook was originally much larger.
For GitHub storage purposes, a smaller sample (`Ecommerce_Sample.csv`) is provided.
The notebook remains the same and will work identically with this sample dataset.
